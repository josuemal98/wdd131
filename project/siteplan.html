Website Site Plan

Wasatch Performance Coatings - Project Plan

Site Name

Name: Wasatch Performance Coatings

Description: This is the company where I work. It is a good name because people in Utah know the Wasatch region, and it clearly shows that we sell high-quality paint and coatings.

Domain: wasatchcoatings.com

Site Purpose

The purpose of this website is to show all our services in one place: automotive paint, industrial coatings, and epoxy flooring. The website will help customers see our products with a clean catalog. It will also have a material calculator tool to help customers find out how many gallons of paint or epoxy they need for their projects.

Target Audience Scenarios

Visitors will use the website to answer these questions:

  • "I have a 400 sq ft garage floor. How many epoxy kits do I need to buy from the store?"
  • "Can I filter the product list to see only automotive paints and tools?"

Color Schema

The colors are selected from our company logo to create a matching brand identity:

Orange (#E65100): This is the main color from our logo. Used for the header background, buttons, and important links.
Light Gray (#F4F6F7): Used for the main page background and section boxes to look clean.
White (#FFFFFF): Used for content containers and grids to make the text easy to read.

Typography

Main Font: Arial / Sans-Serif

Used for all headings, paragraphs, forms, and navigation buttons. It is a standard font that is clean, professional, and easy to read on mobile phones and desktop computers.
